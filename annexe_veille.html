1. Définition et enjeux de la veille informatique

La veille informatique est une démarche proactive et régulière permettant de surveiller l'évolution des technologies, des pratiques et des menaces dans le domaine informatique. Pour un professionnel du secteur IT, elle est indispensable pour maintenir ses compétences à jour et anticiper les changements du marché.

Dans le cadre du BTS SIO, la veille informationnelle est une compétence évaluée au titre du Bloc 6 « Organiser son développement professionnel ».

3. Thèmes surveillés

  • Développement web : évolutions de PHP, JavaScript, frameworks (Laravel, React, Vue)
  • Cybersécurité : nouvelles vulnérabilités, bonnes pratiques OWASP, alertes ANSSI
  • Intelligence artificielle : outils IA pour développeurs, LLM, génération de code
  • DevOps : Docker, intégration continue, automatisation des déploiements
  • ITSM : gestion de parc, GLPI, évolutions des outils d'administration

4. Fiches de veille réalisées

🔒 Les 10 principales vulnérabilités web selon l'OWASP Top 10 2021

L'OWASP Top 10 liste les risques de sécurité les plus critiques pour les applications web. En 2021, les injections SQL, les failles d'authentification et les configurations incorrectes figurent parmi les premières menaces. Cette veille m'a aidé à sécuriser mes développements PHP (PDO, hashage des mots de passe, échappement des sorties).

⚡ L'essor de TypeScript dans les projets JavaScript modernes

TypeScript, le sur-ensemble typé de JavaScript développé par Microsoft, est devenu incontournable dans les projets professionnels. Son adoption a dépassé 60 % des projets Node.js en 2024 selon l'enquête State of JS. Le typage statique réduit les erreurs à l'exécution et améliore la maintenabilité du code.

🤖 GitHub Copilot et les assistants IA pour développeurs en 2025

Les assistants IA de développement comme GitHub Copilot, Cursor ou Claude Code transforment les pratiques de programmation. Ces outils génèrent du code, suggèrent des corrections et expliquent du code existant. Leur utilisation nécessite une pensée critique : le développeur reste responsable de la qualité et de la sécurité du code produit.

🐳 Docker et la conteneurisation dans le développement web

Docker permet d'empaqueter une application avec toutes ses dépendances dans un conteneur portable. En 2024, plus de 70 % des entreprises utilisent des conteneurs en production. Pour un développeur SLAM, maîtriser Docker facilite les déploiements et garantit la reproductibilité des environnements de développement.
