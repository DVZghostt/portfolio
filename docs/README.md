# Dossier de documents — Portfolio BTS SIO

Placez vos fichiers dans les sous-dossiers correspondants à chaque réalisation.

## Structure

```
docs/
├── portfolio/       → Documents liés au portfolio
├── glpi/            → Rapports et captures GLPI
├── checkmystars/    → Livrables CheckMyStars
├── veille/          → Synthèses de veille
├── ocs/             → Documents OCS Inventory
├── dramaltech/      → Documents Dramaltech
├── acaba/           → Livrables ACABA
└── stage/           → Documents de stage / alternance
```

## Formats acceptés

PDF, Word (.doc/.docx), Excel (.xls/.xlsx), PowerPoint (.ppt/.pptx),
Images (.png/.jpg/.jpeg/.gif), Archives (.zip), SQL (.sql), Texte (.txt/.md)
