1. Présentation du projet

Ce projet porte sur le déploiement de OCS Inventory NG pour l'inventaire automatique des postes et logiciels d'un parc informatique. L'installation se fait sur Ubuntu Server 22.04, avec configuration du serveur, de la base de données et des agents clients.

2. Objectifs

  • Installer et configurer le serveur OCS Inventory NG
  • Mettre en place la base de données MariaDB pour le stockage des inventaires
  • Déployer les agents OCS sur les postes clients
  • Documenter précisément la procédure d'installation

3. Environnement technique

  • OS : Ubuntu Server 22.04 LTS
  • Application : OCS Inventory NG
  • Base de données : MariaDB
  • Serveur web : Apache 2
  • Agents : Windows / Linux

4. Contenu de l'annexe

  • Présentation des objectifs de l'inventaire
  • Étapes d'installation détaillées
  • Configuration du serveur OCS et du serveur web
  • Déploiement et configuration des agents
  • Validation de l'inventaire et tests
