1. Présentation du projet

Ce projet consiste en l'installation et la configuration complète de GLPI sur Ubuntu Server 22.04 LTS. Il comprend la mise en place du serveur web Apache, de MariaDB, de PHP 8.1, ainsi que le paramétrage du système de helpdesk et de gestion de parc informatique.

2. Objectifs

  • Mettre en place une solution de gestion de parc et de tickets
  • Installer et configurer les services nécessaires (Apache, PHP, MariaDB)
  • Assurer la sécurité et la disponibilité de l'application
  • Documenter la procédure d'installation et de configuration

3. Environnement technique

  • OS : Ubuntu Server 22.04 LTS
  • Serveur web : Apache 2
  • Base de données : MariaDB
  • Langage : PHP 8.1
  • Application : GLPI

4. Contenu de l'annexe

  • Étapes d'installation détaillées
  • Configuration des services Apache, PHP et MariaDB
  • Création de la base de données et paramétrage du site GLPI
  • Tests et validation du bon fonctionnement
